# 财务报表预测模型

目标|生成财务预测，支持决策 应用|财务计划，credit分析，并购，业务分析和其他 类型|What if question 作用|用模型方便整理思路，减少漏考虑关键因素的可能

# 准备

  • Step1 理解使用这个模型的期望，以及期望的输出

    • 了解为什么要创建这个模型
    • 了解需要什么样的输出
    • 基于这个模型可以做什么样的决策

  • Step2 收集公司的历史数据，行业的以及竞争对手的

    • 开始前至少有一些历史数据, 3到5年以上的数据，基于相同的标准
    • 同时准备一些行业和竞争对手的数据，需要这些数据来做benchmarks判断公司做的好不好
    • 宏观经济数据的预测，如GDP增长率，利率变化趋势等等

  • Step3 理解公司的计划并整理出一套综合的模型假设

    • 这是很关键的一步，很多人跳过了
    • 在不了解计划前，很难做出准确的预测
    • 没有唯一正确的解法，大家常用的线性回归不见得是好方法
    • 如果在历史数据中看到趋势，使用它
    • 销售驱动(Sales Driven)预测， 从销售增长率开始, Bottom Up
    • 策略却动(Pilicy assumptions), Top Down

  • Step4 构建模型并测试

    • 通过一些例子来练习
    • 需要考虑的因素： 折旧(Depreciation),留存收益(Retained Earnings),利息支出(Interest Expense), 签署财务报表的约定和格式需要一致(Sign Convention and Formatting of Financial Statements)

  • Step5 根据反馈改善模型

    • 收集反馈，持续改进

# Uses of the Model

I discussed the uses of the common size statements earlier. Now you can see by looking across each row and the averages which items can be forecast as percentage of sales and which you will need to forecast using some other approach. These statements give you a lot of insight into the company and point out issues you need to investigate. For example, you may want to investigate how the company has managed to bring down its ratio of Cost of Sales to Sales, is this improvement sustainable, is it now in line with the industry, and so on.
